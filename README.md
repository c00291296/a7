# Assembly Airplane's Astonishing Acceleratory Advancement And Annihilation

This is a 3D game, built completely from scratch, in Assembly, for Games Fleadh
2024. Fly, dodge, collect flags and meet your ultimate demise crashing into one
of those ominous pyramids!

!(game poster)[https://github.com/C00291296/motorola_mayhem/blob/airplane/poster.png?raw=true]

Requires [EASy68k](http://www.easy68k.com/) to run. Just open the *stort.x68*
file in the editor and press start!
